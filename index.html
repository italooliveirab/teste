📜 Regras do FIRENET Chip Blast

Objetivo: junte 3 ou mais chips iguais na horizontal, vertical ou diagonal para eliminar e pontuar.

🚀 Chip FIRENET: é raro e funciona como power-up. Quando cai, explode uma área grande ao redor.

Concorrentes: TIM, Claro e Vivo às vezes atrapalham com eventos rápidos, como controles invertidos, lentidão ou chip extra.

🎁 Caixa FIRENET: é rara. Ela cai rápido dentro da arena. Clique nela antes de tocar o chão para ganhar moedas. Se cair no chão, você perde a caixa.

Moedas: servem para desbloquear skins na loja. Não existe compra com dinheiro real.

Controles: arraste para mover, swipe para baixo para queda rápida e toque duplo para acelerar.
